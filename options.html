Tracking, mining, and privacy notes

I want to keep this extension as quick, simple, and easy to use as possible.
To keep your safety and privacy, I'm not using any tracking libraries and 3rd party services.
There's only one sort of tracking for this extension, it's the one that the Chrome Web Store uses to keep track of how many people use this extension. It does not track how you interact with the extension, it's just a number of active users per week.
You can even see this on the extension page: Chrome Web Store

I use no mining or any other sort of resource based harvesting.
If you suspect a website or extension is mining on your computer, use one of these: Mining blockers
Curious about what a miner would do to your computer?
I have an experimental page that can show you this: Experimental Browser Miner using Coinhive

Because there is no tracking, mining, or other privacy invasive things being used, this extension should be quick as possible.
If you have any concerns, please contact me.
